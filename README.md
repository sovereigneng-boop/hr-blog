# 직장인 권리 설명서

Next.js와 Tailwind CSS로 만든 HR 정보 블로그입니다.

## 기술 스택

- Next.js 14 (App Router)
- React 18
- Tailwind CSS

## 주요 기능

- 메인 페이지에 HR 관련 블로그 글 목록 카드 표시
- 글 카드에 제목, 날짜, 카테고리, 짧은 요약, 예상 읽기 시간 표시
- 카드 클릭 시 글 상세 페이지로 이동
- 연차, 퇴직금, 4대보험 관련 샘플 글 3개 포함

## 실행 방법

1. 의존성 설치

```bash
npm install
```

2. 개발 서버 실행

```bash
npm run dev
```

3. 브라우저에서 접속

```text
http://localhost:3000
```

메인 페이지에서 직장인 권리 설명서 글 목록을 확인하고, 카드를 클릭하면 상세 내용을 볼 수 있습니다.
